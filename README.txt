This package is an implementation of the algorithm detailed in:

    Sheng Zhang and Guang Lin (2018). Robust data-driven discovery of
    governing physical laws with error bars. Proc. R. Soc. A. 474:20180305.
    http://doi.org/10.1098/rspa.2018.0305

    Sheng Zhang and Guang Lin (2020). SubTSBR to tackle high noise and outliers
    for data-driven discovery of differential equations. Journal of Computational
    Physics. https://doi.org/10.1016/j.jcp.2020.109962

The files in the folder 'RVM' were originally downloaded from
http://www.relevancevector.com and have been modified.

This package has been tested in Matlab 2019a. Download all the files
and run 'demo.m' for an example.
